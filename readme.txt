Dual Mode Indian Sign Language Communicator with Gesture Recognition

How to Run the Project

1. Download the Project Files  
- Download the project ZIP file and extract its contents.  

2. Install Required Libraries  
- Open a terminal or command prompt in the extracted project folder.  
- Install all required libraries using: pip install -r requirements.txt(required libraries present inside the requirements.txt)

3. Download and Install Additional Dependencies  
The following dependencies must be installed for the project to function properly:

a) Python (Version 3.12.4)  
Download and install Python from the official website:  
[Python 3.12.4](https://www.python.org/ftp/python/3.12.4/python-3.12.4-amd64.exe)

 b) Playsound  
Download and install Playsound:  
[Playsound 1.3.0](https://files.pythonhosted.org/packages/67/8c/c9f46b4b194126c4abb12e96321a6bea5c8dcc5c0e4d97622c14dfabe299/playsound-1.3.0.tar.gz)

c) FFmpeg  
Download and extract FFmpeg:  
[FFmpeg Release Essentials](https://www.gyan.dev/ffmpeg/builds/ffmpeg-release-essentials.7z)  
Make sure to add the extracted FFmpeg folder to the system's environment variables.

d) Git  
Download and install Git:  
[Git for Windows](https://github.com/git-for-windows/git/releases/download/v2.46.0.windows.1/Git-2.46.0-64-bit.exe)

e) JDK (Java Development Kit)  
Download and install JDK:  
[JDK 22](https://download.oracle.com/java/22/latest/jdk-22_windows-x64_bin.msi)  
Ensure the JDK installation path is added to the system’s environment variables.

4. Verify Environment Variables  
After installing the dependencies, check if their paths are present in the system’s environment variables.  
If not, manually add them to the system path.

5. Create a Virtual Environment  
- Navigate to the project folder in the terminal.  
- Run the following command to create a virtual environment: python -m venv .venv
- Activate the virtual environment 

6. Run the Project  
- Start the Django server by running: python manage.py runserver
- Open the displayed URL in a web browser to access the application.

7. Troubleshooting  
If you encounter issues while running the project, ensure that:  
- All required dependencies are installed correctly.  
- Environment variables are set properly.  
- The virtual environment is activated before running the project.  

---

By following these steps, you should be able to run the "Dual Mode Indian Sign Language Communicator with Gesture Recognition" project successfully.
